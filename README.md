# json-converter
This project converts industrial machine telemetry data from two different JSON formats into a single unified format. The objective is to ensure that telemetry data coming from different sources can be processed consistently by downstream systems.
Input Formats
Format 1 (data 1.json)
Format 2 (data 2.json)
It contains:

Device information
Timestamp in milliseconds since epoch
Location stored as a single string
Operational status and temperature
ISO 8601 timestamp

#Technologies Used
Python 3
JSON
Datetime Module
Unittest Framework
